# Wyoming OpenAI Edge TTS

Home Assistant app that packages OpenAI Edge TTS and Wyoming OpenAI into one container for voice pipelines.

It is designed to solve practical streaming support for Edge TTS in Home Assistant by exposing a Wyoming endpoint that delivers responsive, incremental TTS playback.

See `DOCS.md` for configuration details.
